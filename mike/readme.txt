== Description ==

Mike is a tiny WordPress theme for blogs and personal sites. No page builder, no Google Fonts, and a single compact stylesheet.

Built for speed: a clean Mike site scores around 99/100 on Lighthouse mobile out of the box, with no caching plugin, CDN, or tracking code required.

Features:

* Tiny, lightweight theme.
* WordPress 7.0 ready; PHP 7 and 8 ready.
* No page builder. The only script is a tiny optional menu-accessibility helper you can turn off.
* Compact CSS in a single stylesheet.
* Self-hosted fonts via WordPress's Font Library (WordPress 6.5+); system-font fallback on older versions.
* Responsive, looks great on all devices.
* Text or image logo.
* Two menu locations: primary and footer.
* Beautiful typography and editor styles.
* Wide-image and fullscreen image support for writing.
* Translation-ready.

Demo: https://mike-demo.miketran.net

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Mike in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Install' button, then the 'Activate' button to use your new theme right away.
4. Navigate to Appearance -> Customize in your admin panel to customize the theme to your taste.

Full documentation: https://miketran.net/mike/docs/

== Frequently Asked Questions ==

= Does this theme use Google Fonts? =

No. Mike serves self-hosted fonts only, through the WordPress Font Library (WordPress 6.5+). On older versions it falls back to the native system font stack.

= Does this theme load any JavaScript? =

Almost none. The mobile menu opens and closes with pure CSS and small inline handlers. One optional script (js/a11y.js) adds Esc-to-close and focus-trapping to the mobile menu; turn it off in Customize > Misc for a zero-JS-file build.

= How do I remove the theme credit in the footer? =

Customize -> Footer -> turn off "Keep theme credit". GPL does not require keeping it.
